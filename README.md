# 🌦 Weather Forecast App

A modern and responsive weather dashboard that displays real-time weather data for any city.

---

## 🚀 Features
- 🔍 Search weather by city
- 🌡 Real-time temperature updates
- 🌥 Weather conditions display (cloudy, sunny, etc.)
- 📱 Responsive design (works on mobile & desktop)

---

## 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript
- OpenWeather API
